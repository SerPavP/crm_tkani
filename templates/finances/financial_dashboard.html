Справка по финансовой аналитике
Общие показатели (KPI-блоки):
  • Выручка за период: Общая сумма всех оплаченных сделок за выбранный период. Стрелка показывает тренд по сравнению с предыдущим аналогичным периодом.
  • Прибыль за период: Выручка за вычетом себестоимости проданных товаров.
  • Рентабельность: Процентное отношение прибыли к выручке (Прибыль / Выручка * 100%).
  • Средний чек: Средняя сумма одной сделки.
  • Количество сделок: Общее число оплаченных сделок.
  • Топ-ткань месяца: Ткань (среди всех цветов), принесшая наибольшую выручку за последний месяц.
Режимы отображения:
  • Быстрая информация: Краткая сводка ключевых данных на текущий момент (сегодня, неделя, месяц, квартал), а также топ-списки клиентов и тканей.
  • По периодам: Детальная таблица всех сделок за выбранный промежуток времени с общими итогами сверху. Можно выбрать даты вручную или использовать быстрые фильтры.
Топ-списки:
  • Топ-5 клиентов: Можно посмотреть по прибыли, количеству сделок или общей сумме. Клиенты кликабельны, ведут на их детальную страницу.
  • Топ-5 тканей: Можно посмотреть по частоте заказов (сколько раз заказывали), по прибыли или по выручке. Ткани кликабельны, ведут на детальную страницу самой ткани.
